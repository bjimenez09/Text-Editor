## Installation

To Install:

Clone the repository from GitHub (https://github.com/bjimenez09/Text-Editor).

Run these command lines in your terminal to install necessary dependencies:

```
npm install
```

## Usage

Run this command 

```
npm run start
```

Direct your browser to "http://localhost:3000/"
